# 🏠 House Price Prediction (Linear Regression)

A simple machine learning project to predict house prices based on square footage using Scikit-Learn. This project demonstrates basic data modeling and Git version control workflows.

## 🚀 Features
- **Linear Regression Model**: Trained on a sample dataset of square footage vs. price.
- **Train/Test Split**: Evaluates model performance on unseen data.
- **Virtual Environment Support**: Isolated dependencies for clean execution.

---

## 🛠️ Installation & Setup

Follow these steps to get the project running on your local machine:

### 1. Clone the Repository
```bash
git https://github.com/amjadali2705/AI_ML_Assignments.git
cd AI_ML_Assignments
```

### 2. Install Dependencies
```bash
pip install pandas scikit-learn
```

### 3. To train the model and see the prediction results, run:
```bash
python model.py
```

### 4. Example Output
```bash
Model MSE: 0.0
Prediction for 2800 sqft: $560,000.00
```
